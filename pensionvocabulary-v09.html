Pension vocabulary

Version: 0.9

Pension
Information about a person's pension.
endDate
The last date when a temporary pension benefit will be paid. Optional. Only exists for temporary pension benefits.
startDate
The date when the pension is paid to the beneficiary for the first time.
statusCode
Numeric code representing the status of the decision related to the pension benefit. 0 = temporary, 1 = confirmed. Optional. Default value 1.
typeCode
Short code representing the type of the pension benefit. KE = Kansaneläke.
typeName
Human-readable type of the pension benefit, e.g., Kansaneläke.
